# 🛒 Carrefour Retail Analytics Dashboard

An interactive Power BI dashboard providing sales performance, product analytics, customer insights, and operational metrics for Carrefour. This project translates raw retail transaction data into actionable business intelligence to help optimize product placement, improve inventory management, and boost customer satisfaction.

---

## 📌 Project Overview
In the fast-paced retail industry, understanding consumer behavior and product velocity is essential. This repository provides a comprehensive dashboard tracking data from Carrefour supermarket operations. 

By analyzing sales distribution across different product categories, customer membership statuses, payment methods, and daily/monthly trends, this dashboard offers strategic insights for retail managers looking to improve profitability and operational efficiency.

---

## 📂 Repository Structure
The files in this repository are organized as follows:

*   **`Wallpaper/`**: Custom-built background images and custom canvas layouts designed to keep the visual aesthetics professional and readable.
*   **`Carrefour.pbix`**: The primary Power BI desktop file containing the complete dataset, data transformations, calculations, and visual charts.
*   **`Carrefour.pdf`**: A high-quality static PDF report export of the final dashboard for quick presentations or portfolio viewing.
*   **`Carrefour_Data.xlsx`**: The underlying retail dataset used for building the pipeline and analytical measures.

## 🚀 Key Features & Insights
*   **Sales Performance Tracking**: Evaluation of overall revenue, gross margin, and volume trends across various supermarket branches.
*   **Product Performance**: Identification of top-selling categories, worst-performing items, and inventory velocity insights.
*   **Customer Insights**: Segmentation between regular shoppers and loyalty program members, highlighting their average spending habits.
*   **Operational & Transaction Metrics**: Analysis of peak shopping hours, busy weekdays, and preferred payment channels (e.g., Credit Card, Digital Wallet, Cash).

---

## 🛠️ Tools & Technologies Used
*   **Power BI Desktop**: Data modeling, advanced DAX implementation, and building interactive storytelling layouts.
*   **Power Query**: Data preparation, handling data formats, handling duplicates, and columns cleaning.
*   **DAX (Data Analysis Expressions)**: Custom measures for dynamic revenue KPIs, historical sales comparisons, and average invoice sizes.
*   **Microsoft Excel**: Initial exploratory data analysis and data storage.
